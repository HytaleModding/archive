# HytaleModding Archive

This repository is made to hold archived documents, usually transcripts from the ticketing system. 

### Contributions
Due to the small complexity of this, we are not accepting any PRs. If you wish that you could improve this, please open a issue and if we think it's necessary you can submit the PR. If you can't code, that's alright, we will do the work if we think it's necessary. 


To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```

This project was created using `bun init` in bun v1.3.3. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
